# SocketChat


Clone the project by following command or download the project file
```sh
git clone https://github.com/TuserS/SocketChat.git
```

Change directory to SocketChat or type following command
```sh
cd SocketChat
```
Download node modules
```sh
npm install
```
Run the project
```sh
node index.js
```
Open in web browser
```sh
http://localhost:4000/
```
